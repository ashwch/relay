# Code of Conduct

## Our pledge

We pledge to make participation in this project respectful, welcoming, and
productive for everyone.

This repository is currently private, but these expectations apply now and are
intended to remain suitable if the project becomes public later.

## Expected behavior

Examples of behavior that contributes to a positive environment:

- be respectful and constructive
- assume good intent while still being precise about technical risks
- explain tradeoffs clearly
- accept responsibility for mistakes and fix them
- keep feedback focused on the work, not the person

## Unacceptable behavior

Examples of unacceptable behavior include:

- harassment, insults, or personal attacks
- discriminatory language or behavior
- publishing private information without permission
- derailing technical review with hostile or bad-faith comments
- any conduct that would reasonably make collaboration unsafe or unwelcome

## Technical review norm

This project is correctness-oriented. Direct technical feedback is welcome.

The standard is:

```text
be kind to people
be strict about release safety
be clear about evidence
```

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, and other
contributions that violate this Code of Conduct.

For now, because the repository is private, report concerns through the internal
maintainer or security contact for the project. If the repository becomes public,
this file should be updated with a public reporting address.
